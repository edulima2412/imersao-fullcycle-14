- Grafana
- Mongodb

## Instalação

```bash
$ npm install
$ npx prisma generate
```

## Rodando o nestjs

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rodando o nextjs

```bash
# development
$ npm run dev

# production mode
$ npm run prod
```

## Testando o nestjs

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
